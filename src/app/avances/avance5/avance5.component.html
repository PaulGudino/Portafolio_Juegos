Cambio de Juego

El juego original fue reemplazado por otro debido a numerosas deficiencias y lagunas que confundían a los jugadores. Estos vacíos generaban una experiencia de juego confusa y frustrante. Para mejorar la calidad y la comprensión del juego, se tomó la decisión de desarrollar una nueva versión que abordara estos problemas y proporcionara una experiencia más clara y satisfactoria para los jugadores.

¿Cómo se juega?
El juego contiene 19 dados, pueden jugar 2 – 3 personas, cada persona debe tener 6 dados y hay un dado extra que será para iniciar el juego. El dado del inicio se lo debe tirar hasta que salga un elemento. Para decidir el orden habrá 3 cartas enumeradas del 1 al 3 y cada jugador tomará una al azar. Una vez que ya este definido el orden ya se podrá jugar El jugador debe tirar un dado al ring, también puede colisionar con otros dados para cambiar el color visto desde arriba.

Si el dado que el jugador tiró coincide con el mismo color de un elemento visto desde arriba con otros dados, deba seguir las instrucciones de lo que representa ese elemento.
Fuego (Rojo): El jugador debe recolectar los dados que tienen el color del elemento fuego cuando se ven desde arriba, además si aún hay dados en el ring debe tomar uno y termina su turno.
Agua (Azul): El jugador no debe recolectar los dados que tienen el color del elemento agua cuando se ven desde arriba, y termina su turno.
Tierra (Verde): El jugador debe recolectar los dados que tienen el color del elemento tierra cuando se ven desde arriba como un solo bloque por el jugador y se termina su turno, es decir que, si en algún momento le toca volver a tirar, tendrá que lanzar todo ese bloque de dados.
Aire (Blanco): El jugador debe recolectar los dados que tienen el color del elemento aire cuando se ven desde arriba, además debe sacudir el ring por 2s y termina su turno.
Trueno (Amarilo): El jugador debe recolectar los dados que tienen el color del elemento trueno cuando se ven desde arriba, además de que en caso de tener un solo bloque de dados este anula el efecto del elemento tierra, pero si tiene más de un bloque de dados este anula el efecto del bloque mayor y termina su turno. Si el dado que el jugador tiró coincide con el color del comodín, debera seguir las instrucciones del comodín
Comodín (Naranja): El jugador tira el dado y tiene el color del comodín pueden pasar dos escenarios:
En el ring hay dados que tienen el color del comodín: El jugador debe recolectar los dados que tienen el color del comodín, luego toma una carta de los comodines y finaliza su turno.
En el ring no hay dados que tienen el color del comodín: El jugador toma una carta de los comodines y luego debe volver a lanzar.

Si el ring llega a quedar vacío, el siguiente jugador de turno tendrá que tirar todos sus dados en una superficie plana (no el ring), y los dados vistos desde arriba que no tengan repetidos deberán ser tirados al ring y no aplica ninguna de las reglas antes mencionadas. Después de que el jugador haya realizado la acción ya podrá ejercer su turno de juego.
